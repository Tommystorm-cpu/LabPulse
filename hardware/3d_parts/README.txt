Quick note about the case! The big hole in the side that makes room for the SIM and UPS hat ports is big enough for access to most of it!
One issue I found when I actually printed the thing was that some cables can be too chunky to fit into the case and I had to sand the thing down, 
namely the micro-USB port for the SIM hat and the mini-HDMI ports on the pi itself. If you want to avoid this entirely, 
It might be worth tweaking the model to make more room for these things!

Thank you to user Stamos on printables.com that provided the original case that I was able to modify, without this baseline, this wouldn't have happened!
